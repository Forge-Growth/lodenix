Introduction

KnowledgeHub API

A living intelligence layer that turns real-world, real-time data into structured knowledge your AI tools can call on demand — no RAG infrastructure required.

The problem this solves: Every AI tool you build needs current, domain-specific ground truth. Without it, your AI is guessing. KnowledgeHub gives your AI what's actually working in the world right now.

How it works

Zyte crawls curated, high-signal sources per domain on a schedule. An AI scoring layer extracts and ranks only what's valuable. That knowledge is embedded, stored per collection, and served via clean REST endpoints your tools can call with a single POST request.

Your tool sends a query like "cold email openers that get replies" and gets back ranked, structured, current knowledge — ready to inject directly into your prompt. No vector DB to manage, no scraping to build, no staleness to worry about.

Authentication

All API requests must include your API key as a Bearer token in the Authorization header.

Authorization: Bearer kh_live_xxxxxxxxxxxxxxxxxxxx
Keep your key secret. Never expose it in client-side code. Use environment variables.

Request Format

POST body for /v1/knowledge/{domain}:

Parameter Type Status Description
query string required Natural language query for knowledge retrieval
top_k integer optional Number of results to return. Default: 5, Max: 20
recency_bias float optional 0–1. Higher values weight newer knowledge more. Default: 0.6
min_score float optional Minimum relevance score threshold 0–1. Default: 0.4
format string optional "chunks" (default) or "prompt_ready" — pre-formatted for injection

Prompt Injection Pattern

The recommended way to use KnowledgeHub results in your AI tools:

async def build_sales_prompt(user_query):
    # 1. Fetch fresh knowledge
    knowledge = await knowledgehub.query(
        domain="sales",
        query=user_query,
        top_k=5
    )

    # 2. Build context block
    context = "\n\n".join([
        f"INSIGHT: {r['content']}"
        for r in knowledge["results"]
    ])

    # 3. Inject into system prompt
    system_prompt = f"""
You are an expert sales AI.

CURRENT KNOWLEDGE (what's working right now):
{context}

Use the above insights to inform your responses.
Always apply what's actually working, not generic advice.
    """

    return system_prompt
Pro tip: Use format: "prompt_ready" in your request and we pre-format the context block for you — ready to paste directly into your system prompt.
