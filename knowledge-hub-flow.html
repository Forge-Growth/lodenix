System Architecture

From Raw Web
to Smart API

How the Knowledge Hub turns real-world data into living intelligence your tools can call on demand.

01
Ingestion
Data Sources + Zyte Crawler
Zyte scrapes targeted, high-signal sources per domain on a fixed schedule. Not random crawling — curated source lists that are known to contain what's actually working.
Sales forums & communities
Trading news & signals
Content trend trackers
Niche newsletters
X / Reddit threads
Competitor playbooks
# Schedule per domain (not 24/7)
sales_knowledge → every 2h
trading_signals → every 1h
content_trends → every 3h
↓ raw html / text output
02
Processing
Extraction & Cleaning
Strip ads, nav, boilerplate. Extract only the signal — frameworks, tactics, examples, results. Normalize everything into a consistent schema regardless of source format.
Noise removal
Signal extraction
Schema normalization
Source metadata tagging
↓ clean structured text
03
Intelligence
AI Scoring & Enrichment
AI reads every cleaned chunk and decides — is this genuinely valuable or just noise? Scores, tags, deduplicates and ranks by recency + relevance. This is the moat.
Value scoring (0–1)
Intent tagging
Duplicate detection
Recency weighting
Domain classification
↓ scored & enriched chunks
04
Storage
Vector DB + Knowledge Store
Embedded and stored per domain collection. Each crawl cycle prunes stale data automatically — the knowledge stays fresh without growing unbounded.
Vector embeddings per domain
Metadata store (score, date, tags)
Auto-pruning on each cycle
Hybrid search (vector + keyword)
# Collections
collection: "sales" → 2,400 chunks ↻ 2h
collection: "trading" → 1,100 chunks ↻ 1h
collection: "content" → 3,200 chunks ↻ 3h
↓ query → ranked results
05
API Layer
The Endpoint — Your Moat
Clean REST endpoints per domain. Send a query, get back ranked, current, structured knowledge — ready to inject directly into any AI prompt. No RAG infrastructure needed on the caller's side.
POST /v1/knowledge/sales
POST /v1/knowledge/trading
POST /v1/knowledge/content

# Request body
{
  "query": "cold email openers that get replies",
  "top_k": 5,
  "recency_bias": 0.8
}
↓ structured json response
06
Consumer
Your Tools — Grounded in Reality
Any tool you build calls the endpoint, gets fresh knowledge, injects it into the AI prompt. The AI is no longer guessing — it knows what's working right now in the real world.
AI Sales Tool
Calls /sales → gets current objection scripts, openers, closing tactics
Trading Bot
Calls /trading → gets market narrative, sentiment signals, live context
Content Gen AI
Calls /content → gets hooks, formats, angles blowing up this week
Any Future Tool
Add a new domain, add an endpoint. One integration — infinite tools.
